# Vim-IFC

A Vim plugin with syntax highlighting for the Industry foundation classes IFC format.

![Highlight](/img/highlighting.png)
Example in neovim with a dracula theme.

## Tribute

It actually just wraps the highlighting done by [brunopostle](https://github.com/brunopostle) in a plugin

Further information on the awesome [osarch community](https://community.osarch.org/discussion/29/ifc-vim-syntax-highlighting)
